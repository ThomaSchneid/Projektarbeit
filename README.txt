### ISTALATION ###
***
Mit dem Befehl "git clone git@github.com:ThomaSchneid/Projektarbeit.git" kann das Repo lokal geklont werden.
***
Nachdem das Repository geklont wurde, finden sich im Ordner "Programme_exe" 3 .exe Dateien für die einzelnen Anwendungen.
***
Durch einen einfachen Doppelklick auf das entsprechende Programm wird man aufgefordert ein Bild aus einem bereits existierenden 
Bilderpool auszuwählen (man kann natürlich auch jedes andere gewünscht Bild auf dem Rechner auswählen).
***
Im Ordner "Dokumentaion" finden sich die source .tex Dateien zur Dokumentation im PDF Format, welche sich im Hauptvrzeichnis befindet.
***
Der Ordner "Python" enhält alle Code-Files der ausführbaren Programme. 
***
Für alle Betriebssysteme abseits von Windows können die einzelnen Programme mithilfe einer Konsole oder Python IDE über die Code Files gestartet werden.
